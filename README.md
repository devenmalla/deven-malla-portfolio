# 🌐 Deven Malla Portfolio Website

A modern and responsive developer portfolio built with Vite, React, TypeScript, Tailwind CSS, and shadcn/ui. This portfolio showcases my skills, projects, and personal journey in a clean and minimal interface.

---

## 🧰 Technologies Used

This project is built using:

- ⚡ **Vite** – Fast frontend build tool
- ⚛️ **React** – UI library for building interfaces
- 🧑‍💻 **TypeScript** – Static typing for JavaScript
- 🎨 **Tailwind CSS** – Utility-first CSS framework
- 🧩 **shadcn/ui** – Themeable UI components powered by Radix UI

---

## ✨ Features

- ✅ Responsive design for mobile, tablet, and desktop
- 🌙 Light & dark mode toggle
- 📂 Project showcase with clean cards
- 📧 Contact form section (ready to integrate with EmailJS or backend)
- ⚙️ Easily customizable and scalable folder structure
- 🚀 Ready for production deployment with Vercel
